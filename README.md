# M. Muthu Visweswaran — Legal Portfolio Website

Personal legal portfolio website built with HTML, CSS, and vanilla JavaScript.

**Live Site:** https://muthuvisweswaran.github.io

## Features
- Responsive single-page design
- Timeline-based experience section
- Smooth scroll navigation
- Mobile-friendly layout

## Tech Stack
- HTML5 / CSS3
- Google Fonts (Playfair Display + Inter)
- No frameworks — pure HTML & CSS
